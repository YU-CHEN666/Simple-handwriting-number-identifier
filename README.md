# 簡介
 使用OpenCV來獲取手寫數字，搭配預先訓練好的神經網路模型，進行辨識再顯示結果，一個簡易的手寫數字辨識器就此誕生 :+1: :+1: :+1:
# 操作方法
 執行程式後，會有操作說明視窗，請遵循說明操作。
> [!IMPORTANT]
> 在執行鍵盤操作前，請先確定輸入法是**英文模式(打出英文字母)**，這樣程式才會有反應。
